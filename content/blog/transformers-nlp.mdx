# Transformers and NLP: The Language Processing Revolution

## The Transformers Revolution

Transformers have completely transformed natural language processing. In 2017, the paper ["Attention is All You Need"](https://arxiv.org/abs/1706.03762) changed the game forever.

## Why are They so Powerful?

Transformers outperform RNNs and LSTMs in:

- 🚀 **Parallelization**: Process entire sequences simultaneously
- 🎯 **Context capture**: Attention to any position
- 📊 **Scalability**: Work better with more data
- 🔄 **Transfer learning**: Effective pre-training

## Transformer Architecture

### Attention Mechanism

The heart of the transformer is **multi-head attention**:

\`\`\`python
import torch
import torch.nn as nn
import math

class AttentionHead(nn.Module):
    def __init__(self, d_model, d_k):
        super().__init__()
        self.d_k = d_k
        self.query = nn.Linear(d_model, d_k)
        self.key = nn.Linear(d_model, d_k)
        self.value = nn.Linear(d_model, d_k)
    
    def forward(self, x, mask=None):
        # Linear projections
        Q = self.query(x)  # (batch, seq_len, d_k)
        K = self.key(x)
        V = self.value(x)
        
        # Scaled dot-product attention
        scores = Q @ K.transpose(-2, -1) / math.sqrt(self.d_k)
        
        # Apply mask if it exists
        if mask is not None:
            scores = scores.masked_fill(mask == 0, float('-inf'))
        
        # Softmax to get attention weights
        attention = torch.softmax(scores, dim=-1)
        
        # Apply attention to values
        return attention @ V

class MultiHeadAttention(nn.Module):
    def __init__(self, d_model, num_heads):
        super().__init__()
        assert d_model % num_heads == 0
        
        self.d_k = d_model // num_heads
        self.num_heads = num_heads
        
        self.heads = nn.ModuleList([
            AttentionHead(d_model, self.d_k) 
            for _ in range(num_heads)
        ])
        self.output = nn.Linear(d_model, d_model)
    
    def forward(self, x, mask=None):
        # Apply each attention head
        head_outputs = [head(x, mask) for head in self.heads]
        
        # Concatenate outputs
        concatenated = torch.cat(head_outputs, dim=-1)
        
        # Final projection
        return self.output(concatenated)

### Complete Transformer Block

\`\`\`python
class TransformerBlock(nn.Module):
    def __init__(self, d_model, num_heads, d_ff, dropout=0.1):
        super().__init__()
        
        # Multi-head attention
        self.attention = MultiHeadAttention(d_model, num_heads)
        self.norm1 = nn.LayerNorm(d_model)
        self.dropout1 = nn.Dropout(dropout)
        
        # Feed-forward network
        self.ffn = nn.Sequential(
            nn.Linear(d_model, d_ff),
            nn.ReLU(),
            nn.Dropout(dropout),
            nn.Linear(d_ff, d_model)
        )
        self.norm2 = nn.LayerNorm(d_model)
        self.dropout2 = nn.Dropout(dropout)
    
    def forward(self, x, mask=None):
        # Attention with residual connection
        attention_out = self.attention(x, mask)
        x = self.norm1(x + self.dropout1(attention_out))
        
        # FFN with residual connection
        ffn_out = self.ffn(x)
        x = self.norm2(x + self.dropout2(ffn_out))
        
        return x

## Using BERT with Hugging Face

The Transformers library makes everything much easier:

\`\`\`python
from transformers import BertTokenizer, BertForSequenceClassification
from transformers import Trainer, TrainingArguments
import torch

# Load pre-trained model
model_name = 'bert-base-uncased'
tokenizer = BertTokenizer.from_pretrained(model_name)
model = BertForSequenceClassification.from_pretrained(
    model_name,
    num_labels=2
)

# Tokenize text
texts = [
    "This product is excellent, I highly recommend it",
    "Very bad experience, won't buy again"
]

inputs = tokenizer(
    texts,
    padding=True,
    truncation=True,
    max_length=512,
    return_tensors="pt"
)

# Make prediction
with torch.no_grad():
    outputs = model(**inputs)
    predictions = torch.softmax(outputs.logits, dim=1)
    
print("Predictions:")
for text, pred in zip(texts, predictions):
    print(f"{text[:50]}...")
    print(f"  Negative: {pred[0]:.2%}, Positive: {pred[1]:.2%}\n")

## Fine-Tuning for Your Task

### Prepare Dataset

\`\`\`python
from torch.utils.data import Dataset

class SentimentDataset(Dataset):
    def __init__(self, texts, labels, tokenizer, max_length=512):
        self.encodings = tokenizer(
            texts,
            truncation=True,
            padding=True,
            max_length=max_length,
            return_tensors='pt'
        )
        self.labels = torch.tensor(labels)
    
    def __getitem__(self, idx):
        item = {key: val[idx] for key, val in self.encodings.items()}
        item['labels'] = self.labels[idx]
        return item
    
    def __len__(self):
        return len(self.labels)

# Create datasets

## Comparison: BERT vs GPT

| Aspect | BERT | GPT |
|---------|------|-----|
| Architecture | Bidirectional encoder | Unidirectional decoder |
| Objective | Masked Language Modeling | Causal Language Modeling |
| Best for | Understanding, classification | Text generation |
| Context | Bidirectional | Left to right |

## Best Practices

1. **Start Small**: Use distilled models (DistilBERT) for prototypes
2. **Batch Size**: Adjust based on your GPU (use gradient accumulation)
3. **Learning Rate**: 5e-5 is a good starting point
4. **Warmup**: Use warmup steps to stabilize training
5. **Evaluation**: Validate on data the model hasn't seen
